Etiquetas de párrafo

Este es un párrafo. Aquí puedes escribir texto normal.

Este es otro párrafo. Puedes agregar más texto aquí.

Los párrafos son bloques de texto que se utilizan para organizar el contenido en una página web.

Recuerda que los párrafos se separan entre sí por un espacio en blanco.

También puedes usar etiquetas HTML dentro de los párrafos, como negritas o cursivas.
